# 監督用AI向け引継ぎ資料

このディレクトリには、AIアシスタント（Cursor、Gemini Canvas等）がProject Solの開発を継続するために必要な引継ぎ資料が含まれています。

## 📁 ディレクトリ構成

```
docs/ai-handover/
├── README.md (このファイル)
├── PROBLEM_GENERATION_HANDOVER.md  # 問題生成システムの引継ぎ
├── 00_README.md                    # プロジェクト概要
├── 01_PROJECT_OVERVIEW.md          # プロジェクト全体の概要
├── 02_CURRENT_STATUS.md            # 現在の実装状況
├── 03_FUTURE_PLANS.md              # 今後の開発計画
└── 04_REFERENCE_FILES.md           # 参考ファイル一覧
```

## 🎯 使用方法

1. **新規AIアシスタントが参加する場合**:
   - `00_README.md`から読み始める
   - `01_PROJECT_OVERVIEW.md`で全体像を把握
   - `02_CURRENT_STATUS.md`で現在の実装状況を確認

2. **問題生成システムを理解する場合**:
   - `PROBLEM_GENERATION_HANDOVER.md`を参照
   - `docs/technical/自動問題生成アルゴリズム選定と開発指示.txt`も併せて確認

3. **最新の実装内容を確認する場合**:
   - `docs/technical/IMPLEMENTATION_REPORT.md`を参照

## 📚 関連ディレクトリ
